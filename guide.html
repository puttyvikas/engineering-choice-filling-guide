JoSAA Choice Dashboard Guide

Use this page to understand the data source, filters, tables, and column meanings before filling JoSAA choices.

Data Source

The dashboard uses IIT colleges.xlsx, NIT colleges.xlsx, and III-T Colleges.xlsx, processed into analysis_build/combined_choice_analysis.json. It includes only rows where Seat Type = SC and Gender = Gender-Neutral. Female-only supernumerary rows are excluded because the candidate is male.

IIT rows use JEE Advanced SC rank 1948. NIT and IIIT rows use JEE Main SC rank 7316. For NITs, the default applicable quota set is OS rows plus NIT Warangal HS rows, because the candidate's state of eligibility is Telangana.

How To Use

  1. Use Advanced SC Rank for IIT calculations and Main SC Rank for NIT/IIIT calculations.
  2. Choose College first, Branch first, or Safety first.
  3. Use institute type, quota, cutoff buffer, branch, and college dropdowns to select one or many values.
  4. Filter by search text, BTech-only rows, or 5-year Dual Degree rows.
  5. Remove individual filters by clicking active filter chips.
  6. Use presets for common views such as Top IIT brand, Circuit branches, Safe choices, and Cutoff gaps.
  7. Use Add to compare options in the Selected Choices list.
  8. Use Copy list or Download CSV to export selected choices.

Tables

Top Realistic Options
Rows where Closing Rank is greater than or equal to the applicable candidate rank for that institute type.
Worthwhile Reaches
Rows that missed last year's cutoff, but may be worth placing above realistic choices.
Selected Choices
A temporary comparison list created by clicking Add. It does not modify source data. Choices can be removed, copied, or downloaded as CSV.
Branch Mix
Counts visible realistic options by branch group after current filters.

Column Glossary

College
The institute name, shortened for readability.
Type
The institute type: IIT, NIT, or IIIT.
Quota
The JoSAA quota row used. IIT and IIIT rows are AI. NIT rows are OS, plus NIT Warangal HS for Telangana.
Course
The academic program name, such as Electrical Engineering, Chemistry, or Design.
Branch
A simplified branch group used for filtering and comparison.
Opening Rank
The first rank allotted that program in the selected category and gender row.
Closing Rank
The last rank allotted that program in JoSAA 2025 Round 6 for the selected row.
Rank
The candidate rank used for that row. IIT rows use Advanced SC rank; NIT and IIIT rows use Main SC rank.
Margin
Closing Rank minus Candidate Rank. Positive means inside last year's cutoff; negative means a reach.
Gap
The negative margin shown in the reach table.
Cutoff Buffer / Gap
Heuristic label based on margin versus last year's cutoff. Buffer means the candidate rank is inside last year's closing rank; gap means it missed last year's closing rank.
Buffer %
Heuristic cushion score based on cutoff margin. Higher means more cushion versus last year's closing rank, not an official guarantee.
Institute Score
Advisory score for institute reputation used in sorting.
Branch Score
Advisory score for branch demand used in sorting.
Score
The active sorting score based on the chosen priority mode.
Recommendation Note
A short explanation to help interpret the row.

Quick Presets

Top IIT brand
Focuses on older/top IITs and uses college-first sorting.
Circuit branches
Filters to CSE, AI/Data, Math/Computing, Electronics, and Electrical, and enables BTech-only rows.
BTech only
Shows only rows that include Bachelor of Technology.
5Y dual
Shows only rows whose course name includes both 5 Years and Dual Degree.
Safe choices
Shows large and medium cutoff-buffer rows and uses safety-first sorting.
Cutoff gaps
Shows rows that missed last year's cutoff by 300 ranks or less.
Reset filters
Clears all active filters and search text.

Counselling Note

JoSAA choices should generally be ordered by true preference, not probability. Put dream/reach choices above realistic choices, and safe choices below them. JoSAA scans downward and gives the highest available option.

Limitations

This is an advisory tool using JoSAA 2025 Round 6 data. It is not an official prediction. Always verify final choices against official JoSAA 2026 data before submitting.
