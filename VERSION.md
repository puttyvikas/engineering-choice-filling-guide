# JoSAA Choice Dashboard v3

Saved on: 2026-06-06

## Source

- Data sources: `IIT colleges.xlsx`, `NIT colleges.xlsx`, `III-T Colleges.xlsx`
- Processed source: `analysis_build/combined_choice_analysis.json`
- Scope: IIT, NIT, and IIIT; SC; Gender-Neutral rows
- IIT rank basis: JEE Advanced SC rank `1948`
- NIT/IIIT rank basis: JEE Main SC rank `7316`
- NIT default quota: OS plus NIT Warangal HS for Telangana

## Features

- Separate Advanced SC and Main SC rank inputs
- Multi-select institute type, quota, cutoff buffer, branch, and college filters
- Active filter chips with one-click removal
- Quick presets: Top IIT brand, Circuit branches, BTech only, Safe choices, Cutoff gaps, Reset filters
- Search, BTech-only, and 5Y dual degree toggles
- College-first, branch-first, and safety-first sorting
- Top Realistic Options table
- Worthwhile Reaches table
- Selected Choices list with persistent local storage and reordering
- Copy selected choices and Download CSV
- Branch Mix summary
- Browser guide page and README
